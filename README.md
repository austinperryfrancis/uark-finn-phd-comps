# UARK Finance PhD Comps

This repo contains an Obsidian vault for Finance PhD comprehensive exam prep.

Open `Comps Vault` in Obsidian.

## What Is Here

- `Comps Vault/09 - Prompts`: prompts for triaging papers, building paper notes, and synthesizing literatures.
- `Comps Vault/01 - Paper Notes`: templates for paper notes and referee reports.
- `Comps Vault/02 - Literature Maps`: topic maps for organizing papers by literature.
- `Comps Vault/03 - Exam Question Bank`: templates and pages for past exam questions.
- `Comps Vault/04 - Empirical Designs`: methods notes for identification and research designs.
- `Comps Vault/05 - Theory Derivations`: theory derivation templates and starter notes.
- `Comps Vault/06 - Authors and Datasets`: dataset and author reference notes.

## Simple Workflow

1. Start with `Comps Vault/00 - Comps Dashboard/Home.md`.
2. Use `Comps Vault/09 - Prompts/Fast Triage Prompt.md` for a first pass on each paper.
3. Use `Comps Vault/09 - Prompts/Paper Ingestion Prompt.md` for important papers.
4. Add each paper to the relevant literature map.
5. Practice answers in `Comps Vault/08 - Synthesis Drafts`.

The goal is simple: keep the prompts easy to find, keep the paper notes organized, and make the vault useful for closed-book exam answers.
